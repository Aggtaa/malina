В этом каталоге находятся исходники демонов чтения для mppt контроллеров
иных изготовителей.
Строка компиляции на Raspberry Pi:
 gcc -o mpptd-epn -lrt mpptd-epn.c `mysql_config --cflags --libs`

После компиляции в бинарный файл, его следует скопировать как /usr/bin/mpptd
Данные, собираемые с не-МикроАрт контроллеров, могут отличаться.
Некоторые данные могут отсутствовать вовсе, некоторые получаются расчетным путем.

Демон совместим с батарейным монитором в конфигурации:
а) один контроллер
б) МАП + контроллер


Для этого в настройках БАТАРЕЯ указать, что данные по токам получаются по RS232

----------------------------------------------------------------------------
Описание файлов в каталоге:
----------------------------------------------------------------------------

mpptd-ep.c - трекер EP Solar, чтение по порту RS232. В секции define
 установить ID контроллера. Вместо положения ТММ будут указываться в интерфейсе
стадии заряда АКБ: -, F(loat), B(ulk), (E)qualize
mpptd-epn.c - трекер EP Solar,  работа через локальную сеть. В секции define
 установить IP и ID контроллера.

Сделано для модели EP Solar ETracer ET3415N (серия ETX415N)
